# Gallery

DEMO: https://photo-gallery-2.herokuapp.com/

## Stack Technologies

* Next js
* Redux
* Sass
* Dropzone

## First start

1. Install packages using commands `npm i`
2. Run `npm start dev` to start app
3. Open http://localhost:3000/ to view app in the browser
